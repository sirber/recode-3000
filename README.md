# Recode 3000

Rework on Recode, which was in Free Pascal, in go with [Fyne](https://github.com/fyne-io/fyne).
It's cross platform, and using ffmpeg as a backend.

# Development

1. Install depedencies:

See [Getting started](https://docs.fyne.io/started/).

2. Install packages:

```bash
make install
```

Run

```bash
make run
```

# Build

To build for your current OS, run:

```bash
make build
```

a binary will be produced in `./dist/`.
